# Release Notes - v2026.01

## Overview

This release brings significant enhancements to the Microsoft Fabric Extensibility Toolkit with new UI components, improved client SDK capabilities, and critical performance optimizations. This update focuses on expanding developer tools, enhancing memory management, and providing better deployment configuration options.

## üí• Breaking Changes

### Spark Livy Client Changes

- **Async Session Creation**: `createSession` method in SparkLivyClient is now an async operation
- **Migration Required**: Update existing code that uses `createSession` to handle async operation
- **Impact**: Code using SparkLivyClient will need to be updated to use await/Promise syntax

```javascript
// Old synchronous approach
const session = livyClient.createSession(config);

// New async approach
const session = await livyClient.createSession(config);
```

### API Signature Updates

- **List API Changes**: Some list API signatures have been updated to match current Fabric platform implementations
- **Batch to Async**: Batch operations have been converted to async result patterns
- **Migration**: Review and update code using batch operations to use new async patterns
